   Методические указания к лабораторным работам по курсу АРХИТЕКТУРА ЭВМ.
                              кафедра 304

Материалы собранные в этой методичке созданы преподавателями и студентами 304
кафедры МАИ.  Этот сборник состоит из 2-х частей с идентичным содержанием:
ASCII
        В этой части документы в 866 кодовой странице. Их содержимое можно
        просмотреть с помощью любой ДОС программы-редактора или просмотрщика.
        (Norton commander, например). Для корректного отбражения кирилицы на
        экране необходимо загрузить любой экранный русификатор. (KEYRUS.COM,
        rkey.com, vga866.com и т.д) или использовать 866 кодовую страницу. Для
        установки 866 кодовой страницы необходимо внести следующие изменения в
        конфигурацию компьютера:
в config.sys вставить строки
          DEVICE=C:\DOS\DISPLAY.SYS CON=(EGA,,1)
          COUNTRY=007,866,C:\DOS\COUNTRY.SYS
в autoexec.bat
          MODE CON CODEPAGE
          PREPARE=((866) C:\DOS\EGA3.CPI) MODE CON CODEPAGE SELECT=866
          KEYB RU,,C:\DOS\KEYBRD3.SYS
        Если файлы ДОС находятся в директории отличной от C:\DOS,
        то необходимо изменить пути к файлам. Вышеуказанные
        действия верны только для версий ДОС в которых есть файлы поддержки
        кирилических языков.
WINWORD
        metoda.exe - самораспаковывающийся архив arj. Для распаковки просто
        запустите этот файл и он распакуется в текущую директорию
        В этой части документы в формате WinWord for Windows 6.0.  master.doc
        - мастер-документ. В него включены, как поддокументы, все остальные
        файлы сборника. Этот файл можно использовать для распечатки всей
        методички целиком. Также можно открывать документы и по одиночке.
        Шрифты использованные при создании документов используют кодовую
        страницу 1251. Эта кодовая страница используется в русской версии
        Windows или в пан-европейской версии Windows 95. При просмотре этого
        документа в Windows версии с кодовой страницей отличной от 1251 вместо
        кирилицы будут отображатся пустые квадратики. В этом случае надо
        заменить шрифты на шрифты с кирилицей имеющиеся в наличии .  Обычный
        текст:            Times New Roman Cyr Заглавия:                 Arial
        Cyr Листинги программ:        Courier New Cyr Диаграммы и рисунки:
        MS LineDraw и Courier New Cyr Для предотвращения случайного изменения
        файлов документов на них установлен атрибут Read Only. Если есть
        небходимость что-либо отредактировать необходимо снять этот атрибут с
        помощью досовской команды attrib.exe:  attrib *.doc -R Документ
